# ECCN 문의 응답 시스템 (오프라인)

이 프로젝트는 **외부 API 호출 없이** 로컬 데이터/규칙 기반으로 ECCN 문의에 답변하는 간단한 시스템입니다.

## 기능
- 한국어 질의 입력
- 품목 키워드 기반 ECCN 후보 추천
- ECCN 설명, 통제 사유(Reason for Control), 확인 질문 제공
- API/LLM 없이 동작 (로컬 Python만 사용)

## 실행 방법
```bash
python3 main.py
```

## 사용 예시
```text
질문을 입력하세요 (종료: exit): AES 256 암호화 기능이 있는 네트워크 장비 ECCN이 뭐야?

[응답]
- 추천 ECCN: 5A002
- 근거: strong encryption, crypto
- 설명: Information security 장비(암호 기능 포함)로 분류되는 항목입니다.
- 통제 사유: NS, AT, EI
- 추가 확인 질문:
  1) 암호화가 사용자 접근 가능한 기능인가요?
  2) 정부 전용/군사용으로 특별 설계되었나요?
```

## 주의
- 본 시스템은 내부 검토용 보조 도구입니다.
- 최종 판정은 최신 EAR, CCL, 기업 내부 준법 절차 및 필요 시 전문 자문을 통해 확정해야 합니다.
